XPath Syntax:

# Select tag 
//tagName

# Certain tag
//tagName[1]

# Tag with attribute
//tagName[@attributeName='value']

XPath Functions:
contains()
starts-with()
ex. //tagName[contains[@attributeName='value']]

XPath Operators
//tagName[(expression 1) and (expression2)]

Special Characters:
/: selects the child from the node set on the left side of this character
//: specifics that the matching node set should be located at any level within the document
